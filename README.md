# Blinkit Grocery Sales Analysis | Power BI Dashboard

An interactive **Power BI dashboard** built to analyze grocery sales performance for **Blinkit – India’s Last Minute Delivery App**.  
The project focuses on uncovering sales trends, outlet performance, and product-level insights to support data-driven decision-making.

---

## Dashboard Overview

![Main Dashboard](https://github.com/Tejas2906/Blinkit-Grocery-Sales-Analysis/blob/b4426a82e8db783ac94e47ada4ca24609d71c4f3/Blinkit%20images/Dashboard%20Screenshot.jpg)

---

## Project Objective

- Analyze overall grocery sales performance
- Identify high-performing item categories
- Compare outlet performance by **size, type, and location**
- Track sales trends over time
- Understand customer behavior using ratings and visibility metrics

---

## Key KPIs (At a Glance)

- **Total Sales:** $1.20M  
- **Average Sales per Item:** 141  
- **Total Items:** 8,523  
- **Average Rating:** 3.9
